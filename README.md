# Camping-Website

### Description
WebSite for create digital presence by a Camping, with features about the Create and Delete Cabin in rent or Reserv or Cancel for Client

### Feature's
The properly
- Create a New Cabin to the Rent
- Delete a Cabin by the Site

The client
- Register, Login and Logout session
- Make and Cancel your reservation
